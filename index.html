Welcome to the MagicDrawSysML-Modelica Integration GitHub Page.

This project is an implementation of the OMG SysML-Modelica Transformation OMG SysML-Modelica Transformation. This Java-based implementation supports SysML models from MagicDraw SysML. MagicDraw users can use the MagicDraw plugins provided by this project to import/export Modelica models to SysML.

Installation instructions

Instructions to install and execute the roundtrip SysML-Modelica transformation can be found on the Wiki of the GitHub edu.gatech.mbse.mdsysmlmodelica project here.
